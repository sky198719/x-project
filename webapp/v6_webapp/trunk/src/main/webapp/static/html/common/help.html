投资必读
  • 1.理财人在新新贷获得的理财投资收益合法吗？
    理财人在新新贷获得的理财投资收益合法吗？

    新新贷平台上出借人向借款人出借资金，并按照约定利率收取利息，该利率均符合法律法规的规定，受到法律保护。

  • 2.我账户里的资金安全吗？
    我账户里的资金安全吗？

    为保证投资人的资金安全，新新贷采用多管齐下的资金管理制度:

    a.您可以实时查询到资金账户的详情，且资金只能转出到您当前绑定的银行账号；

    b.当您首次提现时，新新贷相关工作人员会拨打您绑定的手机确认您的银行卡信息以及提现操作是否由您本人发出；

    c.内部制定了严格的资金管理流程和完善安全的系统，网站采用业内领先的系统加密及保护技术，24小时监控。

  • 3.我可以投资吗？
    我可以投资吗？

    只要您注册了新新贷，且账户中有一定的资金，即可投资。新新贷友情提示：提现需要实名认证和银行认证。

  • 4.我的资金账户为什么会被冻结，冻结了怎么办？
    我的资金账户为什么会被冻结，冻结了怎么办？

    如果支付密码连续3次输入错误，资金账户就会被冻结，冻结期间账户资金不得提现，不得投标。若要解冻，需要联系客服，提交实名认证信息，再由技术人员对账户解冻。

  • 5. 散标买了以后，没满标的钱怎么处理?
    散标买了以后，没满标的钱怎么处理?

    一般情况下，投资散标，当前的没满标，会冻结当前的已投资资金，等待标满之后，即进入收益状态。

登录认证
  • 1.为什么会登录失败？
    为什么会登录失败？

    登录失败可能是用户名、密码或验证码输入错误。相比用户名和验证码输入错误，密码输入错误的可能性更大。输入密码前请先确定输入模式，是否需要切换大小写等；其次，输入时保证输入内容正确，如果忘记密码，点击密码输入栏后面的“忘记密码？”，再输入用户名和邮箱，通过邮箱取回密码。

  • 2.手机号码认证后可以修改吗？
    手机号码认证后可以修改吗？

    一般情况手机认证后手机号码不可修改。

  • 3.如何让我的账户密码更安全？
    如何让我的账户密码更安全？

    请使用较长、复杂的字母数字组合以提高密码强度，不要用生日等容易被别人猜中的密码；请记住不要与任何人共享您的密码。请尽快完成实名认证，以保证资金提现账户的唯一性。

充值提现
  • 1.网上银行已经扣款，但新新贷账户余额没有增加？
    网上银行已经扣款，但新新贷账户余额没有增加？

    这是因为银行的数据没有即时传输给我们，我们会在第二个工作日与银行对账后给予确认，请再耐心等待一下。

  • 2.网上银行页面打不开，怎么办？
    网上银行页面打不开，怎么办？

    建议使用Microsoft IE浏览器。点击IE的菜单“工具”—“internet选项”—“安全”—将安全中的各项设置恢复到默认级别。

  • 3.我网上银行重复多次付款了该怎么办？
    我网上银行重复多次付款了该怎么办？

    由于您的支付信息银行没有即时传输给新新贷，造成您的银行重复扣款。不过请放心，我们会在第二个工作日和银行对帐， 确认您的汇款后，重复支付款项会直接充值到您的新新贷账户。

名词解释
  • 新新币

    新新币即在新新贷网站可兑换相应现金的虚拟币。新新币兑换人民币的比例为50:1，即50个新新币可兑换1元人民币。目前新新币的获取渠道为新新贷平台不定期推出的各项权益活动，详情请参见各个活动说明。

  • 体验金

    体验金是新新贷推出的一项投资体验活动，帮助用户体验理财并获得收益。体验金本金不可以提现，体验金是新新贷网虚拟借款产品，计息结束后，新新贷收回体验金本金，产生的收益归用户所有（体验金收益的提现及再投资条件，以不同活动的具体规则为准）。

  • 理财人员绑定

    绑定专业的理财顾问，为用户提供经济动态分享、资产配置、理财规划和账户信息快速查询等服务。

  • 红包规则

    新用户注册成功后会获得新手红包或者参与新新贷其他活动可获得，可在红包有效期内投资产品时按照红包说明使用，每个投资项目仅限使用一张，需手动勾选；对应奖励于项目到期后发放，其他规则以使用条件为准。

  • 冻结金额

    冻结金额是指用户指通过平台投资过程中被冻结的资金。一般有两种情况会被冻结：投标，标还未满时，投的金额暂时会被冻结；提现，向平台申请提现，提现金额还未转到银行卡，资金暂时处于冻结状态。

  • 资产总额

    资产总额=可用余额+冻结金额+待收收益(移动端)

  • 收益管理费

    用户投资散标、债权产品，平台收取投资者利息的10%作为收益管理费，具体以实际产品规则解释为准。

  • 同卡进出

    为保障用户资金安全，使用快捷支付的用户需遵循同卡进出的规则。同卡进出是指使用快捷支付的用户，充值的本金提现时将返回到原卡，收益部分可从其他任意绑定卡提现。比如用户是用A卡充值的，本金提现时必须返回A卡，收益部分则可以从A卡或者B卡以及C卡等其他绑定卡提现。
