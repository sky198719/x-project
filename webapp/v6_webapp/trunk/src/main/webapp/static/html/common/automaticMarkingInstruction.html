受托人：新新贷（上海）金融信息服务有限公司（以下简称新新贷）

鉴于委托人为在新新贷平台上实名注册的用户，现拟参与投资新新贷平台开发的自动配标工具（新元宝、月月派等），为优化委托人的用户体验，委托人就自动投标工具操作的相关事项向受托人做出如下授权：

一、委托内容

委托人委托受托人在其先行选择的标的范围内进行自动配标。

二、委托人承诺

（一）委托人承认并认可自动配标所形成的《借款合同》、《债权转让协议》等合同，并自愿承担由此产生的一切法律后果。

（二）委托人上述委托系其真实意思表达。

三、委托期限

本授权期限自委托人与受托人间签署《自动配标委托书》之日起至完全退出相应自动配标工具（或紧急退出）之日止。

四、本委托书经委托人在新新贷平台以线上点击确认的方式签署。
