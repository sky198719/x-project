产品描述及投资说明

1、投资成功后按0%起计算年化收益， 以后逐月增加0%，到0%封顶。
2、该产品0元起投，0元为一个购买单位，单个账户投资上限为0万元。
3、该产品债权将对接平台的优质借款标的。
4、新手购买该产品可以使用新手红包，每次购买仅限使用1个红包，4个红包在使用上不分先后顺序。
收益如何计算？
本产品投资成功后当日开始计息，每30天整体递增一次年化收益率，第一个30天利息为0%，后面逐步递增升息到0%。
期限 历史年化收益

例： 投资步步高升10000元，65天后赎回，则整体历史收益率升为0%，收益=10000 *（65/360）* 0% =0元。
退出方式
1、每个自然月都有一次手动退出的机会，否则默认继续持有。
2、手动退出时，支持部分或全部本金退出。若退出部分本金，需按照0元整数倍退出，退出后剩余本金按加入时间继续计息。
3、退出的本金当天（含当天）起不计息。
如何回款？
1、退出的本金和收益当日自动返还到您的_账户。
2、若未满30天提前退出，则退出部分的本金不结算收益；
*若投资时使用了红包，且未满30天提前退出，相应红包金额将会扣除。
