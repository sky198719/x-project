1、步步高升的优势有哪些？
步步高升同时兼具逐月递增、提取灵活、低门槛、低风险的特点。
逐月递增：历史年化收益率0%起，以后逐月增加0% ，到0%封顶。
存取灵活：只要有闲钱都可存进，每个月都有一次赎回机会，而且赎回金额灵活，保证按需赎回。
低门槛：0元起投，投资到退出过程0费用。
2、我投资了步步高升5个多月了，如果现在退出，收益如何计算？
步步高升是按日计息，逐月增息，按照加入的期限给予对应的年化利率。 根据从投资当日起每满30天判断作为1个月，之后每个月都依此类推（比如投资日期为2016-5-1，则2016-5-30满1个月，6月29日满2个月）； 用户退出时根据退出日期与投资日期之间的天数判断用户持有处于哪个月(M)，您现在退出，M=5，预期总收益 = 投资金额 * 0% * 计息天数 / 360。
3、投资未满30天时退出，我还使用了新手红包，退出的金额怎么返还？什么时候到账？
未满30天时退出是没有收益的，您使用的红包金额会在退出时从本金中扣除，当日将您申请退出的现金部分返还到您的新新贷账户。
4、如果我要退出部分本金，本金和收益什么时候到账？剩余本金如何计息？
步步高升产品支持部分或者全部本金退出。 若您申请退出部分本金，系统会根据您需要退出的本金和退出当日符合的年化利率标准自动计算出这部分本金的收益， 并将这部分本金和收益当日返还到您的新新贷账户。剩余本金将按加入时间继续计息，直到下次退出。
