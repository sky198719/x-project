网贷之家：5年只做一件事又何妨 互联网金融“匠心”故事

2月21日，网贷之家发表题为《5年只做一件事又何妨互联网金融“匠心”故事》的文章，其中写道成立5年以来，_始终坚持公平金融的理念，凭借互联网科技的强大优势，不断完善大数据风控手段，全面搭建数字化金融信息服务平台。回望五年，作为互联网金融的先行者，_不仅为诸多中小微企业插上飞翔的翅膀——解决了融资难的问题，也降低投资门槛，丰富百姓的投资方式，立志成为所有人的“无忧生钱袋”。


_平台成立于2012年2月，目前由_（上海）金融信息服务有限公司运营，公司总部位于上海，数十家分支服务机构遍布全国。我司秉着专业、透明、规范、快乐的核心价值观，为有投资、借款需求的用户提供全方位的金融信息服务，并用公平金融的理念，致力于改善每个人的生活品质。
