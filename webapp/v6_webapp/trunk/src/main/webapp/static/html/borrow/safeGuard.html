还款保障措施

信贷类（30万以下）

借款人到期还款金额为第一还款来源。

还款保障措施包括但不限于：

1.平台自上线至今专注于为中小微企业提供金融信息服务，积累了颇具特色的中小微企业服务经验；

2.通过自身研发的信用评分模型进行综合授信评估、专业的信审团队进行严格的实地考察、与数十家大数据征信机构合作进行多重交叉验证等，筛选优质借款人；

3.监控客户经营状况和还款能力，结合客户信用评级，定期与不定期进行回访；

4.设立质保服务专款计划，平台上的逾期项目根据借款合同享受质保服务。

信贷类（30万以上）

借款人到期还款金额为第一还款来源。

还款保障措施包括但不限于：

1.平台自上线至今专注于为中小微企业提供金融信息服务，积累了颇具特色的中小微企业服务经验；

2.通过自身研发的信用评分模型进行综合授信评估、专业的信审团队进行严格的实地考察、与数十家大数据征信机构合作进行多重交叉验证等，筛选优质借款人；

3.监控客户经营状况和还款能力，结合客户信用评级，定期与不定期进行回访；

4.借款人提供房产抵押，已办妥房产抵押合同、委托公证、收回房产证等相关房产抵押手续；

5.设立质保服务专款计划，平台上的逾期项目根据借款合同享受质保服务。

房贷类

借款人到期还款金额为第一还款来源。

还款保障措施包括但不限于：

1.借款人提供房产抵押，已办妥房产抵押合同、委托公证、收回房产证等相关房产抵押手续；

2.借款额度控制在房屋评估价值的6折以下；

3.设立质保服务专款计划，平台上的逾期项目根据借款合同享受质保服务。

车贷类

借款人到期还款金额为第一还款来源。

还款保障措施包括但不限于：

1.车辆安装GPS追踪器，以便及时处理逾期财产；

2.在车管所办理车辆抵押登记，相关车辆原件资料交由平台保管；

3.借款额度控制在车辆评估价值的7折以下；

4.设立质保服务专款计划，平台上的逾期项目根据借款合同享受质保服务。
